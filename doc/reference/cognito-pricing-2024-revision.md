# Cognito 料金体系（2024年11月改定）

**作成日**: 2026-03-17
**情報源**: [AWS Cognito Pricing](https://aws.amazon.com/cognito/pricing/)

---

## 3ティア制（2024年11月22日〜）

### Lite ティア
旧来のボリューム割引型。既存User Poolのデフォルト。

**直接サインイン / ソーシャルIdPユーザー:**

| MAU範囲 | 単価 |
|---------|------|
| 0 - 10,000 | **無料** |
| 10,001 - 100,000 | $0.0055/MAU |
| 100,001 - 1,000,000 | $0.0046/MAU |
| 1,000,001 - 10,000,000 | $0.00325/MAU |
| 10,000,001以上 | $0.0025/MAU |

### Essentials ティア
新規User Poolのデフォルト。フラットレート。

- $0.015/MAU
- 最初の10,000 MAU無料
- Liteの全機能 + マネージドログイン、パスキー対応、アクセストークンカスタマイズ

### Plus ティア
プレミアムセキュリティ機能付き。

- $0.020/MAU
- 無料枠なし
- Essentialsの全機能 + リスクベース適応認証、漏洩クレデンシャル検出

## SAML / OIDC フェデレーション（全ティア共通）

| MAU範囲 | 単価 |
|---------|------|
| 0 - 50 | **無料** |
| 51以降 | **$0.015/MAU**（一律） |

**重要ポイント:**
- ティアに関係なく一律 $0.015/MAU
- ボリューム割引なし
- 無料枠はアカウントまたはOrganization単位で50 MAU
- MAU = 月間にサインイン・トークンリフレッシュ・サインアップ等の認証操作を1回以上行ったユーザー

## 特記事項

### リンクアカウントの課金
SAMLまたはOIDC IdPからのユーザーをローカルユーザープロファイルにリンクし、そのリンクされたローカルアカウントでのみサインインした場合は、**ローカルユーザーとして課金**される（より安価）。

→ 大規模環境でのコスト最適化に使える可能性があるが、設計が複雑になるため要検討。

### Pre Token Lambda
User Poolトリガーとして実行されるLambdaの課金はCognito料金には含まれない（Lambda実行料金として別途課金）。ただしログイン時のみ実行されるため、通常は無視できるレベル。

### Machine-to-Machine (M2M)
Client Credentialsフローの料金は別体系:
- Essentials: $2.25/1,000リクエスト
- Plus: $3.00/1,000リクエスト

## 試算テンプレート

```
フェデレーションユーザーのみの場合:
  月額 = (MAU - 50) × $0.015
  年額 = 月額 × 12

例: 10,000 MAU
  月額 = (10,000 - 50) × $0.015 = $149.25
  年額 = $1,791

例: 100,000 MAU
  月額 = (100,000 - 50) × $0.015 = $1,499.25
  年額 = $17,991
```
